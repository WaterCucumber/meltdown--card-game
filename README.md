Meltdown - простенькая быстрая игра, задача которой набрать 100 энергии, не превышая 100 температуры или 50 радиации, а иначе будет плавление (meltdown). В игре несколько зон для карт, ограничивая места для них. У каждой карты есть как положительный так и отрицательный эффект.
